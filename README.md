# Reservation System

Reservation system for the semester assignment in DevOps and TDD.

## Goal

Build one project that covers both subjects:

- meaningful domain logic
- test-first development
- automated CI/CD
- Docker and Kubernetes deployment

## Planned stack

- Node.js
- Express
- Prisma
- PostgreSQL
- Jest + Supertest
- Docker
- GitHub Actions
- Kubernetes

## Planned structure

- src/domain
- src/application
- src/infrastructure
- src/api
- tests/unit
- tests/integration

## Current status

Project scaffold initialized. Domain logic will be added through TDD.
